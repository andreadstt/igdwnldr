# Instagram Templates

HTML templates for Instagram-related tools.

## Files

- `ig_reposter.html` - Instagram Reposter interface
  - 2-column layout (input left, output right)
  - Download posts/reels with custom caption
  - Toggle cover image creation
  - Copy caption, open folder, new download

## Features

- Professional dark mode UI
- Real-time progress tracking
- Custom caption template with @username replacement
- Responsive design
