# Twitter Templates

HTML templates for Twitter/X-related tools.

## Files

- `twitter_thread.html` - Twitter Thread Adjuster interface
  - 2-column layout (input left, output right)
  - Split long text into 280-char threads
  - Character counter per thread with color coding
  - Copy individual or all threads

## Features

- Twitter blue theme
- Real-time character counting
- Thread numbering (1/5, 2/5, etc)
- Empty state and error handling
- Responsive design
